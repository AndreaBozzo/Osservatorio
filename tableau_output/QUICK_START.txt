# 🚀 QUICK START - TABLEAU + DATI ISTAT

## IN 5 MINUTI:

### 1️⃣ Apri Tableau
### 2️⃣ Connetti → Microsoft Excel
### 3️⃣ Seleziona un file .xlsx dalla cartella `tableau_output`
### 4️⃣ Clicca "Foglio 1"
### 5️⃣ Trascina:
   - `Time` → Colonne
   - `Value` → Righe
   - `Territory` → Colore
